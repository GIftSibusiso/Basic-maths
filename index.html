Calculator Guide

This calculator allows you to perform basic arithmetic, solve linear and quadratic equations, and find the general formula for a sequence. Follow the instructions below to utilize its features:

Solving for x in Linear Equations

  1. Enter the equation in the form ax + b = c (e.g., 2x + 3 = 7).
  2. Click the sol button.
  3. The solution for x will appear in the result field.

Solving Quadratic Equations

  1. Enter the equation in the form ax^2 + bx + c = 0 (e.g., x^2 - 3x + 2 = 0).
  2. Click the sol button.
  3. The calculator will display the solutions for x.

Finding the General Formula for a Sequence

  1. Input a sequence of numbers separated by commas (e.g., 2, 4, 6, 8).
  2. Click the sol button.
  3. The calculator will provide the general formula (Tn) for the sequence.

Buttons Explanation

  • C: Clears all input.
  • Del: Deletes the last character entered.
  • sol: Solves equations or sequences.
  • X, X²: Used to define variables in equations.
  • =: Calculates results for arithmetic operations.

Examples

  • Linear Equation: Enter 3x + 5 = 11, click sol, result: x = 2.
  • Quadratic Equation: Enter x^2 - 5x + 6 = 0, click sol, result: x = 2 and x = 3.
  • Sequence: Enter 3, 6, 9, 12, click sol, result: General Formula: Tn = 3n.
